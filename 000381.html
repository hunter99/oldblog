Digital Unix V4.0F上整合tomcat与apache

　　apache 2.0.48，Tomcat4.1.29，OS为Digital UNIX V4.0F，mod_jk1.2.5。

1）安装apache

wget http://www.apache.org/dist/httpd/httpd-2.0.48.tar.gz
tar zxvf httpd-2.0.48.tar.gz
cd httpd-2.0.48
./configure --enable-mods-shared=all --with-egd --with-devrandom --enable-so
make
su
make install

2)安装tomcat

wget http://www.apache.org/dist/jakarta/tomcat-4/v4.1.29/bin/jakarta-tomcat-4.1.29.tar.gz
cd $PATH_4_TOMCAT
tar zxvf jakarta-tomcat-4.1.29.tar.gz
mv jakarta-tomcat-4.1.29 tomcat

3)安装mod_jk
wget http://www.apache.org/dist/jakarta/tomcat-connectors/jk/jakarta-tomcat-connectors-jk-1.2-src-current.tar.gz
tar zxvf jakarta-tomcat-connectors-jk-1.2-src-current.tar.gz
cd jakarta-tomcat-connectors-jk-1.2-src
vi common/jk_global.h
定义snprintf为apr_snprintf，vsnprintf为apr_vsnprintf
./configure --with-apxs=/usr/local/apache2/bin/apxs
make
su
make install

4)配置同linux，网站资料很多。
说明：apache的配置文件中，要
#LoadModule auth_digest_module modules/mod_auth_digest.so
否则不能启动apache,如果你的系统能启动。

5)测试并启动
$APACHE_HOME/bin/apachectl -t
$APACHE_HOME/bin/apachectl restart
$TOMCAT_HOME/bin/startup.sh
