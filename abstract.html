基于Apache的FastCGI技术测试及应用研究
The Performance and the Application of FastCGI on Apache Servers



　　摘　　要

　　本文在对FastCGI技术及其实现以及相关技术进行了详细分析的基础上，从程序长度、运行时间、上传数据量、回传数据量、资源数量、单点请求及并行请求七方面，对FastCGI的性能进行测试并同CGI进行比较，得出了在具体应用中FastCGI综合性能是CGI的2-5倍且运行更加稳定的结论。升级CGI是FastCGI最主要的应用，实验系统升级前后性能比较结果令人满意。FastCGI与数据库API结合，可实现跨平台异构数据库共享，也可用来开发基于Web的数据库访问中间件。文章最后介绍了如何用FastCGI开发分布式计算系统。

　　关键词：　服务器　协议　FastCGI CGI

　　Abstract

　　 Seven variables here are used to test the performance of the FastCGI, an improvement of CGI. The experiment data　shows that the FastCGI can run 2-5 times as fast as CGI can. The most common use of FastCGI is to upgrade those CGI- based information distribution system. Test results of 2 demonstrations indicate that it is a wise decision to use FastCGI rather than CGI. It is easy to share data stored in various database on various platform and to develop the web-based database midware while FastCGI and APIs of databases work together. Finally, a tiny distributed computing system is shown in which FastCGI plays a big role.

　　Key Words: Web Server, Protocol, CGI, FastCGI
