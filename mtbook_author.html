作者簡介

本文件作者 Jedi ，本名林克寰，目前就讀於國立台北護理學院聽語障礙科學研究所碩士班，主修聽力學。

作者多年來始終對於網路文化抱持著濃厚的興趣，曾經參與過 GnuPG 、 GPGShell 、 WinPT 、 BkGnuPG 等跟公開金鑰加密法有關的軟體中文化，同時維護了 GPGZilla 站台，持續地推廣網路隱私與身份權的資訊。除此之外也參與了 Perl 學習手冊第三版、 Perl 網路程式設計、 Absolute BSD 等書的翻譯工作，還跟 hlb 一起翻譯了深入親和力 (Dive Into Accessibility) 。

在跟網誌有關的部分，作者曾經四處巡迴演講跟 Blog 及 Movable Type 相關的題目，獨力翻譯完所有的 Movable Type 官方文件，也與 digdog 及 autrijus 一起維護了 Movable Type 的中文化。諸位此刻所閱讀的 Movable Type 完全手冊也完全出自作者一人之鍵盤而成。

作者自己的網誌位於 http://blog.elixus.org/jedi/ ，與本書或 Movable Type 有關的資訊多半也會在此公布；常常有許多人也會在此提出與 Movable Type 相關的疑惑，也幾乎都能在最短的時間內取得回覆。若您真的遇到了問題，也歡迎隨時前來留言。
