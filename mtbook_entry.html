第三章 ─ 管理文章

這一章將介紹 Movable Type 最基本的功能 ─ 管理文章。所謂的「管理」，包括了張貼新文章的上稿功能、修改既有文章的編輯功能，以及刪除功能。我們將在本章分別介紹這些功能，並以簡單的範例來說明各種動作所會產生的結果。

除此之外，我們也會在這一章裡加入一些跟文章有關的資訊；例如，要如何在文件中鍵入特殊的符號、建立超連結、修改文字樣式等，也都會在這一章裡面提到。

讓文章分出層次

當妳的文章數量開始增加，或者是當妳的文章越寫越長之後，如果在主索引頁面裡就列出全部的文章內容，不但會使得頁面長度遽增，讓讀者們必須花更多的時間下載這一頁，還會使得同一個畫面裡擠進太多資訊，反而讓妳的頁面內容難以被閱讀。

Movable Type 把文章內容分成了兩個欄位：主要文章本體和額外文章內容；在預設的模版裡，祇有第一個欄位的內容纔會顯示在索引頁面裡，而第二個欄位的內容則祇會在單篇彙整頁面中出現。這或許會很有幫助，因為妳可以在第一欄裡寫些妳觀察到的有趣現象，然後再第二欄裡纔開始寫些自己的心得感想。這麼一來，讀者們就可以先大致看到妳觀察到了些甚麼，如果他們對這個主題也一樣感興趣的話，纔花更多的時間精力去仔細閱讀另外的部分。或者，以「起─承─轉─合」的文章架構來說，妳大可用第一欄來寫一些能吸引人的東西，然後在第二欄裡則詳細地加以闡述。現在就讓我們看看要如何實做：

超連結與文字變化

Movable Type 提供了各種不同的方法，來幫助妳找出特定的文章；它在編輯選單會列出最新的五篇文章，方便妳迅速地編輯最近寫過的文章；另外在列出及編輯文章畫面中，妳除了可以選擇要列出所有的文章，然後一篇一篇尋找外，也可以用特定條件來加以篩選，也可以使用 Movable Type 內建的全文檢索功能來尋找文章，或者用更複雜的進階搜尋表單來使用像是正規表示式之類的搜尋字串。

上傳檔案

Movable Type 也能夠讓妳上傳檔案。如果這個檔案是圖片的話， Movable Type 還可以讓妳選擇要用嵌入或彈出的效果來呈現；如果這個檔案不是圖片的話，就會建立起一個鏈結，讓讀者們能夠依此鏈結來下載妳所提供的檔案。

上傳檔案的時候要注意一點：要上傳的檔案請盡可能使用祇含有英文字母跟數字的檔名，否則可能會發生預期外的錯誤。以下我們就將分別按照各種不同的狀況，加以說明：

上傳圖片並以嵌入式圖片建立新文章
