　　我一般采用如下方法，相对简单，但可能不符合某些安全方面的要求。仍以bugzilla为例说明。
　　假定bugzilla解压后的目录为/path/to/bugzilla-2.16.3
对于apache的配置文件，检查
AddHandler cgi-script .cgi .pl
要求有.cgi在这行里。
然后，增加如下几行。
Alias /bugzilla/ "/path/to/bugzilla-2.16.3/"
<Directory "/path/to/bugzilla-2.16.3">
Options Indexes MultiViews Includes FollowSymLinks +ExecCGI
AllowOverride Limit
Order allow,deny
Allow from all
</Directory>
重启apache后，就可以在浏览器中输入http://yourhost/bugzilla进行访问了。

　　从你的问题，我猜测你可能将那些cgi移到cgi-bin目录下了。
