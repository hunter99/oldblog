gcc崩溃了

　　强行装了一个gnu-ld，结果gcc崩溃了。：（

　　折腾了一天，终于又装上了，过程如下：
>tar zxvf gcc-3.3.2.tar.gz
>mkdir gc
>cd gc
>./1.sh
>./2.sh
>su
>make install

其中，两个脚本的内容如下：
---------------------1.sh-------------------------------
#!/bin/bash
LD_LIBRARY_PATH=/usr/i18n/lib:/usr/i18n/shlib:/usr/local/lib:/usr/local/BerkeleyDB.3.3/lib:/usr/local/BerkeleyDB.4.2/lib:/usr/netsca
pe/server4/bin/msg/lib;export LD_LIBRARY_PATH
PATH=/usr/users/yanghd/bin:/usr/i18n/bin:/usr/bin:.;export LD_LIBRARY_PATH
ICONVPATH=/usr/local;export LD_LIBRARY_PATH
../gcc-3.3.2/configure

---------------------2.sh--------------------------------
#!/bin/csh
limit stacksize unlimited
limit datasize unlimited
make bootstrap
