關於 MOVABLE TYPE 完全手冊

這一份手冊的雛形是先前我所撰寫的 Movable Type 新手手冊；在經過了一段時間的實際使用後，許多讀者反應了她們所特別需要（而我卻沒有寫到）的資訊；同時藉由幾堂授課之後，又有許多需要進一步釐清的觀念、或更具體的例子，也都加進了這個版本的手冊裡。

實際上，妳不祇可以把這份完全手冊視為新手手冊的延伸版本，更可以把它視為另一個單獨的版本。因為在我原本的寫作計畫中，就籌畫著要連帶地把相關的議題一併放進來；像是如何申請動態網域名稱服務 (DynDNS) 、如何調校網誌組態與設定檔 (mt.cfg) 、如何讓網誌的介面與樣式更具一致性、如何規劃要在頁面上提供的功能、如何讓網誌更具親和力、甚至是如何決定網誌的出版授權協議等。

這些議題在新手手冊中，都因為篇幅的關係而被刪略了；不過現在妳可以在這裡找到。而本書的適用程度則可以說是從新手一路到中、高階使用者；所以無論妳對於 MT 曾經有過多少認識與瞭解，都還是能從中獲得更多有趣的經驗纔是。

照慣例，這份手冊仍舊可以被當作是 Movable Type 使用手冊的導讀，也可以當作是使用 MT 的入門書。除此之外，本書也會稍微多加提及 Windows 平台下施用的注意事項，來讓這套系統能夠更加普及。

本手冊撰寫之時，雖然會一再強調觀念上的要點，但會盡量以簡明而不累贅的文字來描述；講解模版的章節中，也盡可能地附上註解，並且使用高反差的顏色來標出最重要的幾列；同時在大多數的章節最後一段，也附註使用手冊裡相關聯的章節，讓有興趣的讀者可以進一步地察看更多的細節。

各種縮寫字(abbr.)或頭文字(acronym)首次出現的時候均會加上虛線底線，祇需把滑鼠游標移至其上就可以看到更多訊息；圖片也都附上了替代(alt)文字以及圖片說明，讓無法載入圖片的讀者也能大略知道圖片內的重點。本手冊也做了便捷鍵(accesskey)的設計：無論妳正在閱讀哪一頁，祇要按下 Alt-1 （ IE 的使用者還要多按一次 Enter ）就可以回到手冊總目錄。所有的圖片跟條列的步驟也都加上了定位錨(anchor)；圖片按照編號都標上了 <a id="img_NN"></a> ，而條列項目或步驟則標上了 <a id="foobar_NN"></a> ，以方便叫用。

手冊撰寫難免有不完美之處，還請不吝指教；由於手冊撰寫以及抓圖曠日廢時，而軟體更新版本的速度又相當之快，因此本手冊勢必難以保證每一張圖片都取自最新版的軟體畫面。最明顯的差異之一，恐怕就是部分的頁面是以 Phoenix 0.5 開啟後抓的圖，而另一些頁面卻是以 Mozilla FireBird 0.6 開啟後抓的圖。其中 Mozilla FireBird 0.6 正是 Phoenix 改名後的下一版；另外內容裡也混用了分別取自 MovableType 2.63 跟 2.64 版的畫面，這是因為本手冊撰寫一半之後， MovableType 纔改版，為了使本手冊的進度不至於落後太多，凡是使用介面上無太大差異的畫面便不重新抓取了。希望讀者能夠體諒這一點；當然對於有顯著差異的部分，則會盡量取用最新版軟體的畫面。希望這份手冊，多少對於新學乍練的讀者有所幫助。
