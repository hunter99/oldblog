附錄三 ─ 相關資源

以下這份列表列出了在繁體中文世界中，幾個跟 Movable Type 有關的主要資源；妳可以透過這些地方去取得與 Movable Type 相關的大量資訊，或者是提出問題、與人討論：

  • MTBook.net ：本書的線上版本位於 http://mtbook.net/mtbook.html ，放在這個位置的版本將會時時更新、盡可能以最快的速度修正錯誤與瑕疵，以及補充新資訊。
  • MTBook.org ：與本書有關的各種文件幾乎都列在 http://mtbook.org/index.html ，包含了一份相當精簡的「 Movable Type 新手手冊」以及全部的官方文件譯本。
  • 部落格村：由 rex 建立的網誌入口站台，位於 http://www.blog.idv.tw/ ，其中整理了一份部落格建站資源（ http://www.blog.idv.tw/blogresource.htm ），收納了許多人的作品，值得一看。
  • 深入親和力：由我和 hlb 一起翻譯的 Dive Into Accessibility ，位於http://dia.z6i.org ，這份文件相當簡單好讀，卻講了非常多增加網頁親和力的技巧，甚至還列了不少用於各種網誌系統（包括 Movable Type ）的實際範例。事實上許多在此文件內的技巧，都已經被目前版本的 Movable Type 所採用了。
  • 部落格隨便祭：這是由 hlb 架設的討論區，位於 http://ccca.nctu.edu.tw/~hlb/forum/ ，其中有一區就是在討論 Movable Type 。許多在台灣活躍的網誌作者也都會在此出沒。
  • 台灣部落格 B.B.S 討論群組：這是由台灣部落格所維護的討論區，位於 http://bbs.twblog.net/ ，其中跟 Movable Type 相關的討論都在「部落格麻瓜園」裡。
  • 旗標出版公司：本文件的紙本將交付由旗標出版，不習慣閱讀螢幕的朋友可以考慮購買紙本。
