致謝

本書得以完成，必須要感謝許多人。

首先要感謝 ilya 率先讓我知道有 Blog 這玩意兒，他的銳利目光總是趕在眾人之前就先看到未來的趨勢；接著要感謝 hcchien 提供了 http://blog.elixus.org/ 上的空間及 Movable Type 帳號，這纔讓我開始接觸到 Movable Type 。對於懶惰成性的他來說，這種際遇簡直可以說是可遇不可求的。

跟 DEBBY 一起研究各種 Movable Type 模版技巧的經驗可以說是促使我開始翻譯及撰寫這些文件的出發點，若不是她的關係，恐怕我至今還沒有讀完官方手冊；而 keichii 的提議則是讓我決定要附上詳盡圖解的關鍵點，若不是他一再慫恿，這本手冊將停留在完全沒有圖解的階段。

digdog 對於 Movable Type 中文化的熱血與貢獻功不可沒，工具列按鈕上的麥金塔字型更是漂亮無比；而 autrijus 總是抽空幫忙檢查 Movable Type 的源碼瑕疵、幫忙處理本土化相容性、製作安裝檔等，更是令人欽佩！

gugod 所撰寫的 outline2html 對於我在推廣 Movable Type 幫了非常多忙，而一些 CSS 技巧也是跟他討論之後纔有辦法想出來的。

acer 、 ipa 、 Schee 、 zonble 與鄒景平老師的文章提供了許多歷史性及觀念性的重要資訊， hlb 三不五時就丟出來的有趣鏈結則拓廣了我對於 Movable Type 外掛模組以及 CSS 的珍貴經驗，她們的作品可以說是我學習的跳板，讓我能夠站在巨人的肩膀上看世界。

RiO 、 REIKO 、 melissa 與 Barabas 在我生活最艱困的時候伸出了援手，使得這本書的撰寫無後顧之憂。

本書最後終於得以出版紙本，還得感謝旗標的黃昕暐先生識材而積極聯絡，也得感謝 lman 提供了許多跟實際出版有關的經驗。

最後我還要感謝 macpaul 熱情贊助，讓我能夠更換筆記型電腦，以完成本書的後半段撰寫工作。

沒有這些人的貢獻，這份文件就不可能出現在妳的面前；若有人從這份文件受益了，也請把功勞歸於她們，我祇不過是喜歡花時間寫東西的人罷了。
