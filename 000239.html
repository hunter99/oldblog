jalopy安装使用

　　车东和keke都推荐这个，我也试试，以后也可以放纵一下自己的编程，把美化交给jalopy.:)

　　不知什么原因，需要将$JALOPY/docu/src/xdocs/manual/manual.src.xsl一文做如下修改，才能build通过:

<!--  <xsl:if test="$build.time='-1'">
    <xsl:message>ERROR: You must supply the stylesheet parameter "build.time"</xsl:message>
  </xsl:if> -->

　　编译说明在http://jalopy.sourceforge.net/build.html。
　　cosole的安装运行说明在http://jalopy.sourceforge.net/plugin-console.html。

　　试了一下，果然爽！需要注意的是在windows，运行jalopy.bat其中会调用./lcp.bat，这个相对路径导致前一个bat文件不能在别的目录下运行，也就说是，在默认情况下，要在console/bin目录下运行。
　　参考命令如下：

按要求将所需要的程序和文件下载将设置
cd $JALOPYHOME
cd build
修改一下../docu/src/xdocs/manual/manual.src.xsl
修改一下build.properties
ant
ant build-dist-console
cd ../tmp~/console
cd lib
将此目录的jar文件加到CLASSPATH中
cd ../bin
copy几个java文件到这个目录下
./jalopy *.java

OK!
一切都美了！：）
