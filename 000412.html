Jmagick在osf4上的安装

　　终于试出来了。

$cvs -d :pserver:anonymous@jmagick.yeo.nu:/var/cvs/jmagick login

password:JMagick

$cvs -d :pserver:anonymous@jmagick.yeo.nu:/var/cvs/jmagick checkout JMagick

$cd JMagick

$autoconf

$./configure --with-java-includes=/usr/opt/java141/include/alpha

$make all

$make

$cd generated

$sh 1.sh

$cd src/magick

$sh l.sh

$cd ../../

$make all

$make test

$su

#make install


其中，1.sh的内容如下：
cp _DrawInfo.h magick/magick_DrawInfo.h

cp _Magick.h magick/magick_Magick.h

cp _MagickInfo.h magick/magick_MagickInfo.h

cp _PixelPacket.h magick/magick_PixelPacket.h

cp _ImageInfo.h magick/magick_ImageInfo.h

cp _MagickImage.h magick/magick_MagickImage.h

cp _MontageInfo.h magick/magick_MontageInfo.h

cp _QuantizeInfo.h magick/magick_QuantizeInfo.h

cp PixelPacket.h magick/
