OSW:2.Apache配置

　　说实话，Apache Web服务器的配置，我算是比较熟了，可有些功能由于没用到，竟也不知如何配，虽然它很有用。:)

　　至少有两：

　　1、虚拟主机
　　可以通过不同的IP或不同的域名，在一台机器上用一个Apache运行（不是进程，也不是线程，似乎叫任务更合适些吧）虚拟多个主机。

　　这样可以为SDUCS90到3322申请个域名，不用跟以前那样用别人的域名，还要加目录了。
　　
　　我到3322.org为我这个BLOG申请了一个静态域名hedong.3322.org,就拿它作试验了。（我可是有固定IP的，动态IP怎么办还真不清楚。）

　　打开httpd.conf，发现有个例子，如下：

#<VirtualHost *>
#    ServerAdmin webmaster@dummy-host.example.com
#    DocumentRoot /www/docs/dummy-host.example.com
#    ServerName dummy-host.example.com
#    ErrorLog logs/dummy-host.example.com-error_log
#    CustomLog logs/dummy-host.example.com-access_log common
#</VirtualHost>
　　　修改为：
<VirtualHost *>
    ServerAdmin hdyang@istic.ac.cn
    DocumentRoot /usr/mt/MT-2.64-full-lib/blogs/1st
    ServerName hedong.3322.org
    ErrorLog logs/hedong.3322.org-error_log
    CustomLog logs/hedong.3322.org-access_log common
</VirtualHost>

　　　保存退出，重启Apache.在浏览器URL中输入http://hedong.3322.org，OK，页面出现了。但还有点问题，页面中的URL还是指向egroups.istic.ac.cn的。

　　　在MT的管理界面，点击“功能设定”，将其“站台网址”由http://egroups.istic.ac.cn/mt/blogs/1st/改为http://hedong.3322.org/,将“汇整网址”也照此办理，保存设置，然后“系统更新”全部页面。

　　　再到http://hedong.3322.org看，除了回响和"trackback"因为调用CGI而需要原URL外，其它都改过来了。

　　　
　　2、用户目录
　　访问很多大学教授的个人主页，见其URL大都为"学校/系/~人"的形式，我总是搞不清Apache怎么实现，还以为只有NetScape的Web服务器才能行呢。:(

　　今天看到，只要一行的设置就可以了，好象是USER什么的。

　　我给办公室的同事在egroups上建立个人目录，都是用笨笨地办法，手工定义alias和的，还要加上一大堆Options.

　　具体设置如下：
　　1）这个功能是由一个模块mod_userdir实现的，所以必须有mod_userdir.so且在httpd.conf里装入它；
　　2）配置httpd.conf，增加以下几行。

<IfModule mod_userdir.c>
    UserDir public_html
</IfModule>

　　3）到相应的用户目录下，建立目录并置属性755。
#mkdir public_html
#chmod 755 public_html

　　OK，用户目录设置完成了。

参考文献：开放系统世界，200306，p44

　　将具体的实现隐藏起来，使用户不能通过URL就简单地判断出所有用变量或程序，可以用apache的mod_rewrite模块。
　　例如：
---- 隐藏Apache下的某个目录，使得对该目录的任何请求都重定向到另一个文件

< Directory "/usr/local/apache/htdocs/manual/">
options Indexes followsymlinks
allowoverride all
rewriteengine on
rewritebase /
rewriterule ^(.*)$ index.html.en [R=301]
< /Directory >

比对于某一个简单应用可以是：
RewriteRule /news/(\d+)\.html /news/news\.php\?id=$1 [N,I]
这样就把 http://www.chedong.com/news/234.html 映射到了 http://www.chedong.com/news/news.php?id=234

一个更通用的能够将所有的动态页面进行参数映射的表达式是：
把 http://www.myhost.com/foo.php?a=A&b=B&c=C
表现成 http://www.myhost.com/foo.php/a/A/b/B/c/C。
RewriteRule (.*?\.php)(\?[^/]*)?/([^/]*)/([^/]*)(.+?)? $1(?2$2&:\?)$3=$4?5$5: [N,I]
